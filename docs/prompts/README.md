# Reusable Prompts

These prompts are maintained project artifacts, not immutable transcripts.

`site-redesign-fable5.md` intentionally differs from its archived source: its
delivery instructions require an isolated feature branch or worktree, full
verification, independent review, and a pull request. Merging, pushing `main`,
and production deployment remain operator-approved actions.

`codebase-audit-fable5.md` remains analysis-only and forbids repository changes.
